- **CorsixIT-2sp (Headless Test Runner):** Added `--headless-test=<script>` CLI flag support to `TerminalTriage/Lua/app.lua`. When the flag is set: SDL is initialised with `SDL_VIDEODRIVER=dummy` (no real display required), audio is suppressed (`config.audio=false`), the intro movie is skipped, `SDL.wm.setIconWin32()` is not called, and `App:run()` dispatches to the new `App:runHeadlessTest()` method instead of entering the SDL event loop. `runHeadlessTest` loads the named Lua script, calls it with the App instance as argument, and exits 0 (success) or 1 (error). Created `test_runner.bat` which sets `SDL_VIDEODRIVER=dummy` + `SDL_AUDIODRIVER=dummy` before launching, and `headless_tests/smoke.lua` as the first integration test script. 17 new TDD tests covering flag parsing, script execution paths, and config overrides (295 tests passing).
